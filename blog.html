Q1: What are the different ways to select an element in the DOM?

Use getElementById, querySelector, getElementsByClassName, or getElementsByTagName to target elements.

Q2: What is the difference between innerHTML, innerText, and textContent?

innerHTML includes HTML tags, innerText shows visible text, textContent includes all text, even hidden.

Q3: What is event delegation in the DOM?

Event delegation attaches a single listener to a parent to handle events from multiple children efficiently.

Q4: What is event bubbling in the DOM?

Event bubbling is when an event triggers on a child element and propagates up to its ancestors.

Q5: How do you create, add, and remove elements using JavaScript?

Use createElement to create, appendChild to add, and remove or removeChild to delete elements.
